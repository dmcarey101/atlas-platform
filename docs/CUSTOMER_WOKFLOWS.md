# Atlas Customer Workflows

## Core Product Mission

Atlas helps government affairs professionals:

1. Find representation
2. Evaluate representation
3. Monitor competitors
4. Understand lobbying markets
5. Identify opportunities and risks

---

# Workflow 1: Find Representation

## User

Corporate Government Affairs Leader

## Goal

Identify qualified lobbying firms and lobbyists within a target state or region.

## Search Inputs

- State
- Issue Area
- Industry
- Company Type

## Output

- Lobbying firms
- Individual lobbyists
- Active clients
- Estimated costs
- Potential conflicts
- Relevant experience

---

# Workflow 2: Evaluate Representation

## User

Trade Association Executive

## Goal

Compare potential lobbying firms before engagement.

## Output

- Current clients
- Historical clients
- Industry concentration
- Estimated spend
- Geographic footprint
- Competitor relationships

---

# Workflow 3: Monitor Competitors

## User

Government Affairs Team

## Goal

Track lobbying activity of competitors.

## User Action

Create Watchlist

Examples:

- Amazon
- Block
- Coinbase
- Chevron

## Alerts

- New lobbyists retained
- New firms retained
- New state registrations
- Registration expirations
- Significant spending changes

---

# Workflow 4: Monitor Issues

## User

Trade Association Executive

## Goal

Track lobbying activity around specific policy issues.

## Examples

- Artificial Intelligence
- Energy
- Healthcare
- Financial Services

## Output

- Active firms
- Active lobbyists
- Active clients
- New market entrants
- Activity trends

---

# Workflow 5: Market Intelligence

## User

Public Affairs Professional

## Goal

Understand lobbying activity in a state, region, or industry.

## Output

- Largest firms
- Largest clients
- Fastest growing sectors
- New registrations
- Estimated spending trends

---

# Workflow 6: Vendor Discovery

## User

Corporate Government Affairs Leader

## Goal

Find the best lobbying resources in a specific market.

## Output

- Ranked firms
- Ranked lobbyists
- Estimated cost ranges
- Industry expertise
- Potential conflicts
- Existing client relationships

---

# MVP Success Criteria

A user can:

1. Search for representation.
2. Compare representation options.
3. Create watchlists.
4. Receive alerts.
5. Monitor competitors.
6. Monitor issue activity.
